# 🌡️ Temperature Converter Website

## 📌 Project Overview

This Temperature Converter is a simple and interactive web application developed as part of the **AICTE Oasis Infobyte Internship - Level 1 Task 3**.

The application allows users to convert temperature values between **Celsius (°C)**, **Fahrenheit (°F)**, and **Kelvin (K)** quickly and accurately through a user-friendly interface.

---

## 🚀 Features

* Convert Celsius to Fahrenheit and Kelvin
* Convert Fahrenheit to Celsius and Kelvin
* Convert Kelvin to Celsius and Fahrenheit
* Input validation for numeric values
* Responsive design for desktop and mobile devices
* Modern and attractive user interface
* Instant conversion results

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click the `index.html` file or open it in any web browser.
4. Enter a temperature value.
5. Select the temperature unit.
6. Click the **Convert Temperature** button.
7. View the converted results instantly.
